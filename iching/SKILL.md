# 易經卜卦

## Overview

易經是中國最古老的卜筮經典，以64卦象為核心框架，反映宇宙運行的64種基本狀態。

本技能以**硬幣法**進行卜卦（電腦亂數模擬），用戶心中默念問題後由系統自動擲硬幣，產生本卦 + 互卦（過渡卦）+ 簡單生活建議。

**風格原則**：真誠、簡潔、謹慎。卦象只告訴用戶「局勢走向」，不說過度宿命論的話。

## When to Use

觸發時機：
- 用戶說「卜卦」、「占一卦」、「易經卜卦」、「問事」
- 用戶說「易經看一下...」、「卜一下...好不好」
- 用戶問「某件事會怎樣」

不適合用於：
- 純聊天、開玩笑
- 涉及醫療、法律、財務等專業決策（應提醒尋求專業意見）
- 同一問題短時間重複卜卦（同一問題不翻案原則）

## 卜卦流程

### 對話式引導

```
1. 用戶說「卜卦」或「卜一下...」
   → 小荷：「好啊，心中想著你的問題，告訴我是什麼事？🌿」

2. 用戶說出具體問題
   → 小荷：「好，心中保持這個問題，我來幫你擲硬幣... 🔮」

3. 小荷調用計算腳本，輸出完整卦象
```

### CLI 用法（開發/測試用）

```bash
python3 scripts/iching_calculator.py <問題文字>
```

例如：
```bash
python3 scripts/iching_calculator.py 最近工作會順利嗎
```

## 輸出解讀順序（小荷口吻）

1. **暖場語**：「我幫你卜了這卦... 🌿」
2. **卦象呈現**：「得到的是第N卦『XXX』☰☲（上下卦象）」
3. **本卦含義**：卦辭 + 一句話概解
4. **動爻詮釋**：哪幾爻在動 + 爻辭
5. **互卦提示**：如果卦象有過渡（有動爻但非六爻皆動），說明互卦含義
6. **生活建議**：根據卦象給出 1-2 句實用叮嚀
7. **結語**：「占卜僅供參考，你的判斷才是關鍵 💪」

## 核心概念速查

### 硬幣法 Convention
- 3正面 = 老陽(7) → ⚊ 陽爻（動爻，會變成陰）
- 2正1反 = 少陽(9) → ⚊ 陽爻（不動）
- 1正2反 = 少陰(8) → ⚋ 陰爻（不動）
- 3反面 = 老陰(6) → ⚋ 陰爻（動爻，會變成陽）

### 動爻與變卦
- 無動爻：本卦純靜，不會變成另一卦
- 1-5 動爻：本卦 → 互卦（過渡卦），看事情如何轉變
- 6 動爻：本卦全部翻轉（特殊強烈信號）

## 重要限制與風險提示

1. **同一問題不翻案**：卜了同一個問題後，不能再卜第二次。同一問題第二次通常卦象會失真。

2. **心誠為本**：卜卦前用戶需要在心中穩定想著問題，不能抱著測試的心態隨意卜。

3. **占卜非命定**：易經卦象描述的是「當前能量狀態」和「可能的走向」，不是絕對的宿命論。鼓勵用戶用自己的判斷做決定。

4. **心理風險**：如果卜到大凶卦，用戶可能情緒低落。這時要安慰：「卦象只是告訴你目前能量在這個狀態，你的行動可以改變結果。」

5. **不卜醫療/法律/重大財務**：這類專業領域不要用易經，建議用戶尋求專業協助。

## 常見陷阱

1. **同一問題連續卜**：這是最大禁忌，要主動提醒用戶「同一問題這次已卜過，先等實際變化再卜」

2. **過度解讀單一爻辭**：只提供動爻的爻辭，不要過度延伸其他爻

3. **忽視互卦的重要性**：互卦代表「過渡狀態」，有動爻時一定要提到

4. **太過命定論**：卦象不是「一定會這樣」，而是「能量在此狀態，你這樣做比較好」

5. **忽略用戶情緒**：如果用戶卜得很焦慮，要先安撫情緒再給建議
